# Intro

This repo is a companion for the ["Modern E2E testing with Cypress"](https://slides.com/dobromir/modern-e2e-testing-with-cypress/#/) talk

Please reffer to https://github.com/dobromir-hristov/cypress-e2e-testing-examples for more examples.
## How to run
To run the tests, just do:

```bash
npm install

npm run test
```
